**対象:** `hida_unified_v2.py`
**改訂日:** 2026年6月11日
**改訂者:** とまと（レビューと修正案: Claude Fable 5）
**位置づけ:** 理論（五層構造・sync_type・主導追随の発想・14予測）には一切手を触れない。v2.0実装が理論の主張を支えきれていなかった3点を、実装側から理論に追いつかせる改訂である。

## 修正1: sync_scoreの約分問題（新規指摘）

### 問題

v2.0実装の同期スコアは次の組み合わせだった:

```
coherence = clip01(follower_mean / leader_strength)
raw_sync  = leader_strength × coherence
```

主導層は定義上「全層の最大活動量」であるため、常に `follower_mean <= leader_strength` が成り立ち、clipは作用しない。よって代数的に:

```
raw_sync = leader_strength × (follower_mean / leader_strength) = follower_mean
```

**主導強度はスコアに一切寄与していなかった**（閾値ゲートとsync_typeのラベル決定にのみ使用）。「主導強度×追随整合性」という式は、実質的に「追随層3層の活動量平均」と恒等であり、v1.1の対称的スコアの変種に過ぎなかった。これは「v2.0のスコアはv1.1を積の形に書き換えたものと機能的に等価になりうる」というOpus 4.8改訂3の懸念の、懸念ではなく証明だった。

### 修正

- 追随整合性を主導強度で正規化しない絶対量＋時間遅れ応答の合成に変更（修正3参照）
- スコアを幾何平均に変更: `raw_sync = sqrt(leader_strength × follower_coherence)`

幾何平均は「どちらか一方だけ高くてもスコアは上がらない」性質を持ち、「主導層の暴走だけでは同期にならない」という論文§4.3.5の意図を保ったまま、両因子が独立に寄与する。これにより**予測11（主導強度と追随整合性の独立性）が初めて実装上で検証可能になった**。`leader_strength` と `sync_coherence` は独立の観測量として出力される。

### 閾値の再較正

幾何平均はv2.0の値域よりスコアが大きく出るため、ON/OFF閾値を 0.03/0.02 → **0.08/0.05** に再較正した。較正基準は「同一シナリオ・同一シードでv2.0と同等の意識ON頻度を維持すること」とし、基準シナリオ（赤好き・通常エネルギー・seed42）で v2.0: 2/17ステップON ＝ v2.1: 2/17ステップON（同一のL2スパイクイベントで点灯）を確認した。

## 修正2: 非対称性を測定から力学へ（新規指摘）

### 問題

v2.0では各層が固定順序（L1→L2→L3→L4）で更新されるだけで、主導層が追随層を実際に「引き込む」機構が存在しなかった。非対称性はsync_typeというラベルとして測定側にのみ存在し、力学には存在しなかった。時間遅れ相関や伝達エントロピーを導入しても（Opus 4.8改訂3）、測るべき因果チャネル自体がなかった。

### 修正

引き込み機構（entrainment）を導入。ステップtで主導層が確定すると、ステップt+1の追随層の更新ゲインが上昇する:

```
gain(追随層) = 1 + ENTRAIN_K × leader_strength   （ENTRAIN_K = 0.5、初期DNA）
gain(主導層) = 1.0（自分で自分を引き込まない）
```

ゲインの各層への作用（いずれも既存の更新式の増分のスケーリングであり、新しい挙動は追加していない）:

| 層 | ゲインの作用 | 解釈 |
|---|---|---|
| L1 | エネルギー消費・疲労蓄積の増幅 | 身体動員 |
| L2 | 情動増分・誤差由来の価値変動の増幅 | 情動応答感度 |
| L3 | 予測誤差magnitudeの増幅 | 誤差検出感度 |
| L4 | activation_count増分の増幅 | 記憶活性化感度 |

これにより「主導→追随」の時間的因果が初めて力学として実在する。

## 修正3: 追随整合性への時間遅れ成分の導入（Opus 4.8改訂3への対応）

追随整合性を二成分の合成に変更:

```
coherence = (1 − LAG_WEIGHT) × 同時応答成分 + LAG_WEIGHT × 時間遅れ応答成分
LAG_WEIGHT = 0.5（初期DNA）
```

- 同時応答成分: 追随層3層の活動量平均（絶対量、正規化なし）
- 時間遅れ応答成分: 前ステップの主導信号強度に対する、その追随層の今ステップでの活動上昇比

「たまたま同時に活発」と「主導に引き込まれた」の区別は時間遅れ成分が担う。修正2の引き込み機構があるため、この成分には測定すべき実体が存在する。

## 検証: アブレーションテスト

`test_entrainment_ablation.py` を新規追加。同一シナリオ・同一シード群30本で ENTRAIN_K=0（引き込みOFF）と 0.5（ON）を比較した結果:

| 指標 | OFF | ON | 判定 |
|---|---|---|---|
| 主導イベント直後の追随層活動上昇（平均） | 0.0154 | 0.0174 | ON > OFF |
| 時間遅れ相関 r(主導強度t, 追随平均t+1) | −0.110 | −0.031 | ON > OFF |
| シード単位の勝率（ON > OFF） | — | 25/30 | 偶然なら p ≈ 0.0003 |

**正直な注記:** 効果量は小さい。これは検証環境が静的なグリッドワールドであり、追随層が応答できるイベント（予測誤差・情動増分）自体が疎であるためである。効果を大きく見せるためにENTRAIN_Kを盛ることはしていない。「方向が一貫して正しいこと」がここでの主張のすべてであり、効果量の検討はより豊かな環境での検証課題とする。また、時間遅れ相関は両条件とも負であり（活動量がスパイク後に減衰する系では自然な挙動）、主張できるのは「引き込みONで因果方向の相関が一貫して改善する」ことまでである。

## 互換性

- v2.0の出力（sync_score / sync_type / sync_level / is_conscious）はすべて維持
- 新規出力: `leader_strength`, `sync_coherence`, `last_activities`, `entrain_gain`
- 既定モード・loopモードの回帰確認済み（ゴール到達、例外なし）
- 依存は引き続きPython標準ライブラリのみ、計算量はO(1)のまま

## 残課題（v2.2以降）

- L1の引き込みは現在エネルギー消費の増幅のみ。外受容感度（注意）の変調は身体モデルの拡張が必要
- 時間遅れ応答は1ステップ遅れのみ。複数ステップの伝播（位相同期・伝達エントロピー）はOpus 4.8改訂3の通り将来課題
- sync_typeの離散化問題（Opus 4.8改訂6）は本改訂の対象外。`last_activities` を公開したので、主導度の連続分布への拡張の足場はできている
