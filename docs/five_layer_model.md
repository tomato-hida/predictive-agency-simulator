# 人間の内面運動理論：5層構造

---

## 第1層：身体層（Body Layer）

### 定義
外界の物理現象を電気信号に変換する層。まだ「感じて」いない。

### 構成
- 入力（五感）：光、音、圧力、化学物質 → 電気信号
- 出力：筋肉、声帯、表情筋など

### 実装で分かったこと
- 初期値ゼロはありえない（DNAが初期設定を与える）
- 変化が必要（同じ刺激の繰り返しでは上位層が育たない）

### この層の限界
- 「赤い」「痛い」「甘い」はまだない（ただの電気信号）

---

## 第2層：クオリア層（Qualia Layer）

### 定義
電気信号パターンを認識し、言語でラベル付けする層。記憶と連携して「感じ」を作る。

### 何が起きるか
```
第1層：繰り返し現れる電気信号パターン
第2層：パターンを認識
親が教える：「これは赤いね」
記憶（第4層）：パターン + 「赤い」を保存
次回：同じパターン → 記憶参照 → 「赤い」

これがクオリア
```

### なぜ圧縮が必要か
- 電気信号のまま → 情報量が膨大
- 言葉でラベル化 → 効率的に扱える
- 記憶に保存 → 次から瞬時に認識

### 複雑なクオリア = 組み合わせ
- 複数のラベルの組み合わせ → 名前をつける → さらなる圧縮
- 言語 = 圧縮技術

### この層の限界
- ラベルをつけるだけ
- パターンは見えない（第3層の仕事）

---

## 第3層：構造化層（Structuring Layer）

### 定義
クオリアの系列からパターンを見つけ、予測モデルを作る層。

### 何が起きるか
- バラバラのラベル → パターンとして構造化
- 繰り返しを検出 → ルール形成

### 予測モデル
- 過去のパターンから次を予測
- 予測が当たる → 自動処理
- 予測が外れる → 予測誤差が発生

### 言語による構造強化
- 言語化されたルール → より強固なパターン

### この層の限界
- パターンを見つけるだけ
- 「私」はまだない

---

## 第4層：記憶層（Memory Layer）

### 定義
過去のクオリアとパターンを蓄積し、「私」を形成する層。

### 記憶がないと「私」がない
- 記憶なし → 今この瞬間だけ → 自己がない
- 記憶あり → 経験の蓄積 → 「私」が形成される

### 環境と自己の強さ
- 単純な環境 → 「私」がはっきりする
- 複雑な環境 → 「私」がぼやける

### 実装で分かったこと
- 記憶は自己形成に必須

### この層の限界
- 蓄積するだけ
- 4層が同期しないと意識は創発しない

---

## 第5層：意識層（Consciousness Layer）

### 定義
第1〜4層が同期したときに創発する現象。それ自体が第5層。

### 第5層とは
- 第5層 ≠ 何かの役割を持つ層
- 第5層 = 1-4層が同期した状態そのもの

### 同期の強さ
- 弱い同期 → ぼんやりした意識
- 普通の同期 → 普段の意識
- 強い同期 → 集中状態
- 連続的な強い同期 → ゾーン/フロー（上限70%）

### 予測誤差の役割
- 予測誤差が大きい + 生存に関わる → 同期を強める → 意識がはっきりする

### ブレーカー
- 予測誤差が限界（閾値）を超える → 過負荷 → 気絶
