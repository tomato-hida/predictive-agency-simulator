# 人間の内面運動理論：5層構造

---

## 第1層：身体層（Body Layer）

### 定義
外界の物理現象を電気信号に変換する層。まだ「感じて」いない。

### 構成
- 入力（五感）：光、音、圧力、化学物質 → 電気信号
- 出力：筋肉、声帯、表情筋など

### 実装で分かったこと
- 変化が必要（同じ刺激の繰り返しでは上位層が育たない）

### この層の限界
- 「赤い」「痛い」「甘い」はまだない（ただの電気信号）

---

## 第2層：クオリア層（Qualia Layer）

### 定義
電気信号パターンを認識し、ラベル付けと評価を行う層。記憶と連携して「感じ」を作る。

### クオリアの構成
```
クオリア = パターン + ラベル + 評価値

パターン：繰り返し現れる電気信号のまとまり
ラベル：言語による名前（「赤い」「痛い」など）
評価値：快/不快の度合い（生存に有利か不利か）
```

### 評価値とDNA
```
評価値には、DNAによる初期設定がある
初期値ゼロはありえない（生まれた時点で感受性がある）

例：
- 痛みに敏感な人 / 鈍感な人
- 快に敏感な人 / 鈍感な人
- 極端な初期値 → 痛みと快の混同（自傷行為、激辛好きなど）
```

### クオリアの形成過程
```
第1層：繰り返し現れる電気信号パターン
第2層：パターンを認識
親が教える：「これは赤いね」
記憶（第4層）：パターン + 「赤い」+ 評価値を保存
次回：同じパターン → 記憶参照 → 「赤い」

これがクオリア
```

### なぜ圧縮が必要か
- 電気信号のまま → 情報量が膨大
- 言葉でラベル化 → 効率的に扱える
- 記憶に保存 → 次から瞬時に認識

### 複雑なクオリア = 組み合わせ
- 複数のラベルの組み合わせ → 名前をつける → さらなる圧縮
- 言語 = 圧縮技術

### この層の限界
- ラベルと評価をつけるだけ
- パターンは見えない（第3層の仕事）

---

## 第3層：構造化層（Structuring Layer）

### 定義
クオリアの系列からパターンを見つけ、予測モデルを作る層。

### 何が起きるか
- バラバラのラベル → パターンとして構造化
- 繰り返しを検出 → ルール形成

### 予測モデル
- 過去のパターンから次を予測
- 予測が当たる → 自動処理（意識不要）
- 予測が外れる → 予測誤差が発生

### 言語による構造強化
- 言語化されたルール → より強固なパターン

### この層の限界
- パターンを見つけるだけ
- 「私」はまだない

---

## 第4層：記憶層（Memory Layer）

### 定義
過去のクオリアとパターンを蓄積し、「私」を形成する層。

### 記憶がないと「私」がない
- 記憶なし → 今この瞬間だけ → 自己がない
- 記憶あり → 経験の蓄積 → 「私」が形成される

### 環境と自己の強さ
- 単純な環境 → パターンが繰り返される → 「私」がはっきりする
- 複雑な環境 → パターンが分散 → 「私」がぼやける

### 実装で分かったこと
- 記憶は自己形成に必須
- 記憶がない状態では自己（self_strength）が育たない

### この層の限界
- 蓄積するだけ
- 4層が同期しないと意識は創発しない

---

## 第5層：意識層（Consciousness Layer）

### 定義
第1〜4層が同期したときに創発する現象。それ自体が第5層。

### 意識とは何か
```
意識 = 予測誤差に対応するための全層協調処理モード

発動条件：
1. 予測誤差が大きい
2. かつ、生存に関わる

この条件を満たすとき、1-4層が同期し、
リソースを集中投入する処理モードが発動する。
これが「意識」。
```

### 第5層の本質
- 第5層 ≠ 何かの役割を持つ独立した層
- 第5層 = 1-4層が同期した状態そのもの

### 同期の強さ
- 弱い同期 → ぼんやりした意識
- 普通の同期 → 普段の意識
- 強い同期 → 集中状態
- 連続的な強い同期 → ゾーン/フロー

### 意識の持続率（70%問題）
```
実装で判明した事実：
- 集中環境でも意識の持続率は約70%
- 100%にはならない
- これはバグではなくフェイルセーフ

理由：
- 意識 = リソース集中モード
- 100%持続 = 脳が過負荷
- 自動的に30%はオフになる（保護機能）
```

### 意識の閾値（0.3）
```
予測誤差が0.3を超えると意識が発動
この閾値は環境が変わっても一定
```

### ブレーカー（気絶）※要検証
```
予測誤差が閾値を大きく超える
→ 全層が同時にMAX活性化
→ 処理容量を超過
→ ブレーカーが落ちる（気絶）

これは「徐々に弱まる」ではなく「瞬時に落ちる」

※理論的予測。実装による検証は未実施。
※機械はびっくりしない（一気にメモリを食いつぶすほどの予測誤差が作れない）
```

---

## まとめ

| 層 | 名前 | 役割 |
|---|---|---|
| 1 | 身体 | 物理現象 → 電気信号 |
| 2 | クオリア | パターン + ラベル + 評価値 |
| 3 | 構造化 | パターン発見、予測モデル |
| 4 | 記憶 | 蓄積、「私」の形成 |
| 5 | 意識 | 1-4層の同期状態そのもの |

### 意識の本質
- 意識は「予測誤差 × 生存重要度」で発動する
- 意識は「全層協調処理モード」である
- 意識は持続率70%で自動制御される
- 意識は閾値0.3で発動し、過剰な予測誤差で停止する（気絶）
