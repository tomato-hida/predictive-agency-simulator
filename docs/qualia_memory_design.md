# クオリアと記憶の力学 設計図

---

## 物理クオリア

### 構造
```python
physical_qualia = {
    'signal_pattern': [0.7, 0.2, ...],  # 電気信号パターン
    'dna_value': -0.9,                   # DNA初期値（出発点）
    'learned_value': -0.5,               # 学習による上書き
    'current_value': -0.5,               # 現在の値
    'label': '痛い'                      # 言葉
}
```

### 生成条件
1. 電気信号パターンが繰り返し現れる
2. 親が言葉を教える「これは痛いね」
3. パターン + DNA値 + 言葉 → 物理クオリア完成
4. 記憶（第4層）に保存

### 更新ルール
```
current_value = dna_value + 学習による変化

例：コーヒーの苦味
  DNA: -0.8（苦い=危険）
  学習: +1.2（飲むと目覚める=良い）
  現在: +0.4（好き）
```

### 特徴
- センサー固定 → 数は増えない
- DNA値は初期値として保持（リセット可能性）
- 学習で上書き可能
- 病理治療、トラウマ克服が可能
- 完全にゼロにはならない（DNA痕跡は残る）
- 言葉が変わると別物扱い（「痛い」→「痛み」）

## 感情クオリア（レベル1：基本）

### 構造
```python
basic_emotional = {
```

### 生成条件

2. 親が言葉を教える「怖いね」
3. DNA値 + 言葉 → 基本感情クオリア完成

### 特徴

## 感情クオリア（レベル2以降：複雑）

### 構造
```python
complex_emotional = {
    'components': [
        ('fear', 0.6),      # 基本感情への参照
        ('discomfort', 0.4) # 別の基本感情
    ],
    'label': '不安',
    'value': 計算値,
    'created_at': 時系列番号
}
```

### 生成条件
1. 新しい経験をする
2. 複数の基本感情が同時発火
3. その組み合わせに名前をつける
4. 組み合わせ + 言葉 → 複雑感情クオリア完成
5. 記憶に保存

### 計算式
```
value = Σ(component_value * weight)
```

### 再帰的構造
```
レベル3の感情 = レベル2の組み合わせ + 言葉
レベル4の感情 = レベル3の組み合わせ + 言葉
...理論上は無限に増える可能性
```

### 実際の制約
- 人生で経験できることは有限
- 「人生同じことの繰り返し」
- 新しい組み合わせは減っていく
- 大人になると感情クオリアの増加は鈍化

年齢による変化：
```
10代：急速に増える（初体験多い）
20-30代：やや増える（新しい経験まだある）
40代以降：ほぼ増えない（経験済みのパターン）
```

例外（新しい感情クオリアが生まれる）：
- 大きな環境変化（結婚、転職、病気）
- 新しい趣味・文化への没入

### 特徴
- 時系列で増加
- 経験依存
- 個人差が大きい
- 理論上無限、実際は数百〜数千程度

## 記憶

### 構造
```python
memory = {
    'image': [...],                         # 映像データ
    'label': 'あのときの夕日',             # 言葉
    'physical_qualia': ['赤い', '暖かい'], # 物理クオリア配列
    'emotional_qualia': ['寂しい', '安心'], # 感情クオリア配列
    'timestamp': 記憶形成時刻
}
```

### 生成パターン
1. 単独記憶（映像のみ）
2. 単独記憶（感情のみ）
3. 単独記憶（物理クオリアのみ）
4. 複合記憶（全部折り重なる）

### アクセス方法
- 言葉で検索: '夕日' → 該当記憶を返す
- 映像で検索: 類似パターン → 該当記憶を返す
- クオリアで検索: '寂しい' → 該当記憶を返す

### 特徴
- 映像 + 言葉 + 物理クオリア + 感情クオリアが折り重なる
- 折り重なりパターンは無限
- 同じ要素を共有する記憶同士が繋がる

## 層間の相互作用

### 第1層 → 第2層
```
電気信号パターン → 物理クオリア生成
```

### 第2層 ⇄ 第4層
```
クオリア → 記憶に保存
記憶 → クオリア参照（「あ、これ赤いだ」）
```

### 第4層での感情クオリア生成
```
経験 → 複数クオリア発火 → 記憶に保存
名前をつける → 新しい感情クオリア登録
次回 → 記憶参照で呼び出し
```

### 重要な力学
- 物理クオリアは増えない（入力固定）
- 感情クオリアは増える（組み合わせ無限、実際は有限）
- 記憶が増える → 使えるラベルが増える
- ラベルが増える → 世界の解像度が上がる
