# クオリアと記憶の力学 設計図

---

## 入力の分類

### 全ての入力は物理クオリアになる
```
入力（電気信号）
├── 外部入力: 光、音、触覚、味、匂い...
└── 内部入力: 乳酸、ホルモン、心拍、血糖値、体温...
        ↓
    全て物理クオリアとして処理
```

**重要：** 内部身体状態（乳酸、ホルモン等）も電気信号として第1層に入る。外部と内部の区別は入力元の違いだけで、処理は同じ。

---

## 物理クオリア

### 構造
```python
physical_qualia = {
    'signal_pattern': [0.7, 0.2, ...],  # 電気信号パターン
    'source': 'external',                # 'external' or 'internal'
    'dna_value': -0.9,                   # DNA初期値（出発点）
    'learned_value': -0.5,               # 学習による上書き
    'current_value': -0.5,               # 現在の値
    'label': '痛い'                      # 言葉
}
```

### 生成条件
1. 電気信号パターンが繰り返し現れる（外部 or 内部）
2. 親が言葉を教える「これは痛いね」
3. パターン + DNA値 + 言葉 → 物理クオリア完成
4. 記憶（第4層）に保存

### 更新ルール
```
current_value = dna_value + 学習による変化

例：コーヒーの苦味
  DNA: -0.8（苦い=危険）
  学習: +1.2（飲むと目覚める=良い）
  現在: +0.4（好き）

例：乳酸（内部入力）
  DNA: -0.5（不快）
  学習: +0.3（運動した証拠=達成感と結びつく）
  現在: -0.2（やや不快だが許容）
```

### 特徴
- センサー固定 → 数は増えない
- DNA値は初期値として保持（リセット可能性）
- 学習で上書き可能
- 病理治療、トラウマ克服が可能
- 完全にゼロにはならない（DNA痕跡は残る）
- 言葉が変わると別物扱い（「痛い」→「痛み」）

## 感情クオリア（レベル1：基本）

### 構造
```python
basic_emotional = {
    'trigger_qualia': ['heart_rate_up', 'sweat'],  # 発火条件となる物理クオリア
    'initial_value': -0.7,                          # 初期値（由来は問わない）
    'current_value': -0.7,                          # 現在の値
    'label': '怖い'                                 # 言葉
}
```

### 入力 = 物理クオリア
感情クオリア（基本）は物理クオリアの組み合わせで発火する。

```
心拍上昇（物理）+ 発汗（物理）→「怖い」（感情・基本）
乳酸蓄積（物理）+ 筋肉痛（物理）→「疲れた」（感情・基本）
```

### 生成条件
1. 特定の物理クオリアの組み合わせが発生
2. 初期値が発火（DNAか早期学習かは問わない）
3. 親が言葉を教える「怖いね」
4. 物理クオリアパターン + 初期値 + 言葉 → 基本感情クオリア完成

### DNAについての注記
- 痛み、空腹、温度 → DNA由来（確実）
- 快/不快の基本傾向 → DNA由来（ほぼ確実）
- 「怖い」「悲しい」等の感情 → DNA由来か学習かは専門外
- **実装では「初期値がある」とだけ仮定し、由来は問わない**

### 特徴
- 物理クオリアの組み合わせで発火
- 初期値あり（由来は問わない）
- 学習で値が変化する可能性あり

## 感情クオリア（レベル2以降：複雑）

### 構造
```python
complex_emotional = {
    'components': [
        ('fear', 0.6),      # 基本感情への参照
        ('discomfort', 0.4) # 別の基本感情
    ],
    'label': '不安',
    'value': 計算値,
    'created_at': 時系列番号
}
```

### 生成条件
1. 新しい経験をする
2. 複数の基本感情が同時発火
3. その組み合わせに名前をつける
4. 組み合わせ + 言葉 → 複雑感情クオリア完成
5. 記憶に保存

### 計算式
```
value = Σ(component_value * weight)
```

### 再帰的構造
```
レベル3の感情 = レベル2の組み合わせ + 言葉
レベル4の感情 = レベル3の組み合わせ + 言葉
...理論上は無限に増える可能性
```

### 実際の制約
- 人生で経験できることは有限
- 「人生同じことの繰り返し」
- 新しい組み合わせは減っていく
- 大人になると感情クオリアの増加は鈍化

年齢による変化：
```
10代：急速に増える（初体験多い）
20-30代：やや増える（新しい経験まだある）
40代以降：ほぼ増えない（経験済みのパターン）
```

例外（新しい感情クオリアが生まれる）：
- 大きな環境変化（結婚、転職、病気）
- 新しい趣味・文化への没入

### 特徴
- 時系列で増加
- 経験依存
- 個人差が大きい
- 理論上無限、実際は数百〜数千程度

## クオリアの全体像

```
第1層（電気信号）
├── 外部入力（光、音、触覚...）
└── 内部入力（乳酸、ホルモン、心拍...）
        ↓
第2層（物理クオリア）
├── 外部由来:「赤い」「痛い」「甘い」...
└── 内部由来:「疲れた感じ」「ドキドキ」...
        ↓
第2層（感情クオリア・基本）← 物理クオリアの組み合わせで発火
├──「怖い」「悲しい」「嬉しい」...
        ↓
第2層（感情クオリア・複雑）← 基本感情の組み合わせ + 言葉
├──「不安」「切ない」「懐かしい」...
└── 成長とともに増加
```

## 記憶

### 構造
```python
memory = {
    'image': [...],                         # 映像データ
    'label': 'あのときの夕日',             # 言葉
    'physical_qualia': ['赤い', '暖かい'], # 物理クオリア配列
    'emotional_qualia': ['寂しい', '安心'], # 感情クオリア配列
    'timestamp': 記憶形成時刻
}
```

### 生成パターン
1. 単独記憶（映像のみ）
2. 単独記憶（感情のみ）
3. 単独記憶（物理クオリアのみ）
4. 複合記憶（全部折り重なる）

### アクセス方法
- 言葉で検索: '夕日' → 該当記憶を返す
- 映像で検索: 類似パターン → 該当記憶を返す
- クオリアで検索: '寂しい' → 該当記憶を返す

### 特徴
- 映像 + 言葉 + 物理クオリア + 感情クオリアが折り重なる
- 折り重なりパターンは無限
- 同じ要素を共有する記憶同士が繋がる

## 層間の相互作用

### 第1層 → 第2層
```
電気信号パターン（外部+内部）→ 物理クオリア生成
```

### 物理クオリア → 感情クオリア（基本）
```
物理クオリアの組み合わせ → 感情クオリア発火
例：心拍上昇 + 発汗 → 「怖い」
```

### 第2層 ⇄ 第4層
```
クオリア → 記憶に保存
記憶 → クオリア参照（「あ、これ赤いだ」）
```

### 第4層での感情クオリア生成
```
経験 → 複数クオリア発火 → 記憶に保存
名前をつける → 新しい感情クオリア登録
次回 → 記憶参照で呼び出し
```

### 重要な力学
- 物理クオリアは増えない（入力固定）
- 感情クオリアは増える（組み合わせ無限、実際は有限）
- 記憶が増える → 使えるラベルが増える
- ラベルが増える → 世界の解像度が上がる
