# HIDA Explorer：主観的な世界を持つエージェント

## 概要

前方1マスしか見えないエージェントが、首を回して周囲を見渡し、**主観的な内部マップ**を構築しながら探索・タスク実行する。

## L3: 予測誤差検出

見た瞬間に記憶と比較：
```
  ⚡ 予測誤差 @ (5, 3)
     予測: object → 現実: empty
  💭 「あれ？あったはずなのに...」
```

人間の「違和感」と同じ：
- 記憶:「コップここにあったはず」
- 見る:「ない」
- →「あれ？」

## Split-brain構造

```
右脳（行動）: BFS、ルールで決まる（確実）
左脳（作話）: なぜそうしたか後付けで説明

→ ollamaが間違えても行動に影響なし
→ ただ喋るだけ
```

## 神の介入テスト

```python
# 途中でボールを動かす
👁️ 神の介入：ボールを動かす！
（HIDAはまだ知らない...）

# HIDAがボール位置に近づくと...
⚡ 予測誤差 @ (5, 3)
   予測: object → 現実: empty
💭 「あれ？あったはずなのに...」

# 再探索して発見
🎯 ボール再発見！
🎊 ミッション成功！
```

## ollamaを有効にする

`narrator.py` の `USE_OLLAMA = True` に変更

## 実行

```bash
python main.py
```
